Installation du projet avecYarn & Grunt

Step
Object

4#Object.

Créer un objet pour contrôler l'applicationsources/bundle.js

$ make object

$(document).ready( ... );
Permet UNIQUEMENT d'initialiser des objets.

log()
Affiche des informations en console si debug est à true.
/!\ Utilisée à chaque début de fonction, elle permet de contrôler l'efficacité et les performances du code.

init()
Initialise l'objet en orchestrant le lancement des différentes méthodes.
/!\ Ne contient que des appels de fonctions et des vérifications pour leur déclenchement.

Variables

5#Variables.

Placer les variables globales dans l'objetsources/bundle.js

$ make variables

Périmètre d'une variable

  • La position d'une variable dépend de son utilisation.
  • Deux variables qui ont le même périmètre d'utilisation peuvent être regroupées dans un même sous-objet.

/!\ Tips: la position des fonctions est similaire à celle des variables pour permettre de construire des "modules" indépendants et réutilisables facilement.

8#Optimisation.
